<ul>

</ul>

<p><em>Example:</em></p>

<h3>
<a id="user-content-problem-2-correct-brackets" class="anchor" href="#problem-2-correct-brackets" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 2. Correct brackets</h3>

<ul>
<li>  Write a JavaScript function to check if in a given expression the brackets are put correctly.</li>
</ul>

<p><em>Example of correct expression:</em> <code>((a+b)/5-d)</code>.
<em>Example of incorrect expression:</em> <code>)(a+b))</code>.</p>

<h3>
<a id="user-content-problem-3-sub-string-in-text" class="anchor" href="#problem-3-sub-string-in-text" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 3. Sub-string in text</h3>

<ul>
<li>  Write a JavaScript function that finds how many times a substring is contained in a given text (perform case insensitive search).</li>
</ul>

<p><em>Example:</em></p>

<p>The target sub-string is <code>in</code></p>

<p>The text is as follows:
We are liv<strong>in</strong>g <strong>in</strong> an yellow submar<strong>in</strong>e. We don't have anyth<strong>in</strong>g else. <strong>in</strong>side the submar<strong>in</strong>e is very tight. So we are dr<strong>in</strong>k<strong>in</strong>g all the day. We will move out of it <strong>in</strong> 5 days.</p>

<p>The result is: <code>9</code></p>

<h3>
<a id="user-content-problem-4-parse-tags" class="anchor" href="#problem-4-parse-tags" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 4. Parse tags</h3>

<ul>
<li>
<p>You are given a text. Write a function that changes the text in all regions:</p>

<pre><code>&lt;upcase&gt;text&lt;/upcase&gt; to uppercase.
&lt;lowcase&gt;text&lt;/lowcase&gt; to lowercase
&lt;mixcase&gt;text&lt;/mixcase&gt; to mix casing(random)
</code></pre>
</li>
</ul>

<p><em>Example:</em> We are <code>&lt;mixcase&gt;living&lt;/mixcase&gt;</code> in a <code>&lt;upcase&gt;yellow submarine&lt;/upcase&gt;</code>. We <code>&lt;mixcase&gt;don't&lt;/mixcase&gt;</code> have <code>&lt;lowcase&gt;anything&lt;/lowcase&gt;</code> else.</p>

<p><em>The expected result:</em><br>
We are LiVinG in a YELLOW SUBMARINE. We dOn'T have anything else.</p>

<p><em>Note: Regions can be nested.</em></p>

<h3>
<a id="user-content-problem-5-nbsp" class="anchor" href="#problem-5-nbsp" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 5. nbsp</h3>

<ul>
<li>  Write a function that replaces non breaking white-spaces in a text with <code>&amp;nbsp</code>;</li>
</ul>

<h3>
<a id="user-content-problem-6-extract-text-from-html" class="anchor" href="#problem-6-extract-text-from-html" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 6. Extract text from HTML</h3>

<ul>
<li>  Write a function that extracts the content of a html page given as text.</li>
<li>  The function should return anything that is in a tag, without the tags.</li>
</ul>

<p><em>Example:</em></p>

<pre><code>&lt;html&gt;
  &lt;head&gt;
    &lt;title&gt;Sample site&lt;/title&gt;
  &lt;/head&gt;
  &lt;body&gt;
    &lt;div&gt;text
      &lt;div&gt;more text&lt;/div&gt;
      and more...
    &lt;/div&gt;
    in body
  &lt;/body&gt;
&lt;/html&gt;
</code></pre>

<p><em>Result:</em> Sample sitetextmore textand more...in body</p>

//str = '1, Pesho, 1 text Gosho'
