Problem 1. JavaScript literals

Array literal: Gosho,Pesho,Stancho

Comma in array literal: Gosho,,,,,,Pesho

Decimal integer literal: 10

Hexadecimal integer literal: 161

Octal integer literal: 8

Floating number literal: 1.5

Boolean literal: true

Object literal: Pesho Petrov Age: 25

String literal: Pesho

Problem 2. Quoted text

Double quoted string: "How you doing?", Joey said.

Single quoted string: 'How you doing?', Joey said

Problem 3. Typeof variables

Typeof(10) = number

Typeof(10.5) = number

Typeof(true) = boolean

Typeof(['pesho', 'gosho']) = object

typeof(undefined_variable) = undefined

Problem 4. Typeof null

Type of nullable = object

Type of undefined = undefined

Typeof(function()) = function
